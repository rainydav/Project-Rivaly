# Rivaly

Проєкт створений студентами ЗПФК.

## Про наш проєкт

**Rivaly** — це проєкт для проведення турнірів, тестів, вікторин тощо.

У цьому проєкті реалізовано:

- робочу систему реєстрації та входу в акаунт;
- перевірку ролей користувачів;
- зміну функціоналу відповідно до ролі користувача;
- створення турнірів;
- створення тестів;
- створення вікторин;
- створення команд.

## Як запустити проєкт

Для запуску проєкту потрібно ввести в консолі такі команди:

### Запуск backend

bash
cd ../rivali/backend
node server.js

cd ../rivali/Front-end
npm run dev
